✦ AskFriday

Hover a Discord message, ask AI to draft your reply.

AskFriday is a Vencord plugin. Hover any message, click the Ask Friday button, and an LLM drafts a reply — using the surrounding conversation as context — straight into your compose box. You read it, edit it, and hit send. It never sends for you.

View on GitHub Read the docs

What it does

  • Drafts a reply to any message into your compose box (you stay in control of sending).
  • Reads the recent conversation around the message so replies follow the thread.
  • Works with Claude, ChatGPT, and Gemini — via an API key, or your existing subscription through the official CLI.
  • Tone presets: human, casual, technical, professional, witty, robotic, or your own.
Desktop only — Vencord on the desktop app or Vesktop. It won't run on Discord in a browser.

Install

Requires Node.js, pnpm, and git. AskFriday is built into Vencord from source.

# 1. get a Vencord source tree
git clone https://github.com/Vendicated/Vencord && cd Vencord
pnpm install --frozen-lockfile

# 2. drop AskFriday into userplugins (copy, don't symlink)
mkdir -p src/userplugins
cp -r /path/to/AskFridayDiscord/askFriday src/userplugins/askFriday

# 3. build and inject into your client
pnpm build
pnpm inject   # pick Vesktop / your desktop Discord

Fully restart Discord, then Settings → Vencord → Plugins → AskFriday, enable it, and open the cog to set your provider, key, and tone. Full steps in the README.
